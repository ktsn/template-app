# template-app

[![Greenkeeper badge](https://badges.greenkeeper.io/ktsn/template-app.svg)](https://greenkeeper.io/)

A project template for web apps

## Development Setup

```bash
# Dependencies
$ npm install

# Dev server
$ npm run dev

# Test with watch
$ npm run dev:test

# Test & lint (single run)
$ npm test
```
